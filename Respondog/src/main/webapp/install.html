LMS Admins: How to Install ResponDog with your LMS

ResponDog is certified to comply with 1EdTech specifications for LTI Advantage Complete (version 1.3) including Dynamic Registration.

Does your LMS support LTI Dynamic Registration?
  • IF YES: the LMS administrator should go to the LTI registration page in your LMS and use the Dynamic Registration URL: https://respondog.com/registration
    Complete a brief form, activate the ResponDog app, and configure which organizational units in your institution can access it. That's all!

  • IF NO: please select your LMS from the following options for detailed installation instructions

Installation instructions for Blackboard Learn

The institution's account administrator should perform the following steps:
  1. Browse to https://respondog.com/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  2. Go to System Admin | Integrations: LTI Tool Providers | Register LTI 1.3 Tool
  3. Enter the Client ID: be1004de-6f8e-45b9-aae4-2c1370c24e1e
  4. Make a copy of the deployment_id and set Tool status: Approved
  5. Institution Policies: Send Role, Name, Email; Allow Grade Service and Membership Service
  6. Submit
  7. Use the tokenized link from the email in step 1 to register your deployment_id with ResponDog. If the link expired, you can simply repeat step 1 to get another one.
  8. Go back to the LTI Tool Providers page, and from the dropdown menu on the ResponDog app select Manage Placements
  9. Click Create Placement
    • Label: ResponDog
    • Description: ResponDog is an Open Educational Resource for engaging audiences with in-class polls.
    • Handle: (any unique string)
    • Availability: Yes
    • Deep Linking Tool (do not allow student access)
    • Target Link URI: https://respondog.com
    • Custom Parameters (leave blank)
    • Submit
The course instructor can then create any ResponDog poll:
  1. Go to the course | Content | Build Content | ResponDog
  2. Enter a title for the poll and submit

Installation instructions for D2L Brightspace

First of all, Brightspace supports LTI Advantage Dynamic Registration! The Brightspace documentation can be found here. We strongly encourage you to try that first (see above), but the institution's account administrator still has the option of manual registration:
  1. Browse to https://respondog.com/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  2. In Brightspace, ensure the Manage LTI Advantage Tool Registrations and Manage LTI Advantage Tool Deployments permissions are set at the organization level.
  3. From the Admin Tools menu, click Manage Extensibility. In the LTI Advantage tab, click Register Tool.
  4. Choose Standard registration, click Register.
  5. Tool Name: ResponDog
  6. Description: ResponDog is an Open Educational Resource for college-level General Chemistry
  7. Domain: https://respondog.com Note: Domains can only be used in one enabled registration at a time. If you attempt to register a tool with a domain that is already in use, you will be prompted to disable the existing registration.
  8. Redirect URIs: https://respondog.com
  9. OpenID Connect Login URL: https://respondog.com/auth/token
  10. Target Link URI: https://respondog.com
  11. LTI version: LTI 1.3
  12. Keyset URL: https://respondog.com/jwks
  13. Tool URL: https://respondog.com
  14. Extensions: Deep Linking, Assignment and Grade Services, Names and Role Provisioning Services
  15. Deep Linking Content Selection URL: https://respondog.com
  16. Under Roles, choose to Context role information. Sending Institutional role information is optional. Setting the Deployment security to Anonymous will work; however, instructors will be unable to verify or synchronize ResponDog scores to the Brightspace grade book using this setting.
  17. Substitution Parameters: resource_link_id_history=$ResourceLink.id.hisory
  18. Custom parameters: (none)
  19. Click Register
  20. Review the configuration details to obtain the client_id and deployment_id, and click the tokenized link in the email confirmation from step 1 to register them with ResponDog. If the link expired, you can simply repeat step 1 to get another one.

Installation instructions for Instructure Canvas

The institution's account administrator should perform the following steps:
  1. Browse to https://respondog.com/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  2. Configure a new LTI Developer Key for your Canvas Account (see detailed instructions here)
  3. Use the following Key Settings:
    • Key Name: ResponDog
    • Owner Email: admin@respondog.com
    • Redirect URIs:
      https://respondog.com
    • Configure Method: Enter URL
    • JSON URL: https://respondog.com/registration?UserRequest=config
  4. Click Save
  5. Copy or write down the client_id and deployment_id created in step 2. This is the tricky part, because Canvas doesn't make it easy. Canvas uses the developer key as the client_id, so it can be viewed from the list of developer keys. It is a numeric value that looks something like 32570000000000041. The deployment_id can be found in Settings | Apps | App Configurations by opening the settings menu for ResponDog. It is a compound value that consists of a number and a hex string separated by a colon and looks something like 10408:7db438070728c02373713c12c73869b3af470b68.
  6. Add ResponDog as an External App to your account using the client_id created in step 2 (see detailed instructions here)
  7. Click the tokenized link in the email confirmation from step 1 to register the client_id and deployment_id with ResponDog. If the link expired, you can simply repeat step 1 to get another one.
The course instructor can then create any ResponDog assignment as follows:
  1. Create a new Canvas assignment with the following recommended parameters:
  2. Name: (as appropriate, e.g. Class Poll - War of 1812)
  3. Points: 5, or as many questions as your poll will have
  4. Submission Type: External Tool
  5. External Tool URL: Find ResponDog or enter https://respondog.com
  6. Click continue from the ResponDog popup window, then click the red 'Select' button in Canvas
  7. Save or Save and Publish
  8. When you launch the assignment, you may use the links on the Instructor page to customize it for your class.

Installation instructions for Moodle

First of all, Moodle supports LTI Advantage Dynamic Registration! The Moodle documentation can be found here. We strongly encourage you to try that first (see above), but the institution's account administrator still has the option of manual registration:
  1. Browse to https://respondog.com/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  2. As an admin user, go to "Site administration > Plugins > External tool > Manage tools"
  3. Click the link to "Configure a tool manually"
  4. External Tool Configuration
    • Tool Name: ResponDog
    • Tool URL: https://respondog.com
    • Tool Description: ResponDog is an Open Educational Resource for teaching and learning college-level General Chemistry
    • LTI version: LTI 1.3
    • Public Key Type: Keyset URL
    • Public Keyset: https://respondog.com/jwks
    • Initiate Login URL: https://respondog.com/auth/token
    • Redirection URIs:
      https://respondog.com
    • Custom parameters: (none)
    • Check 'Supports Deep Linking'
    • Content Selection URL: https://respondog.com
    • Services | LTI Assignment and Grade Services: select Use for grade sync only
    • Services | LTI Names and Role Provisioning: select Use this service
    • Privacy | check Force SSL
    • Save Changes
  5. When you have finished the configuration, Moodle generates a preconfigured tool. You must activate it and then click 'View configuration details'. When you have these details in hand, including the client_id and deployment_id, click the tokenized link in the email confirmation from step 1 to register the configuration details with ResponDog. If the link expired, you can simply repeat step 1 to get another one.
The course instructor can then create any ResponDog assignment as follows:
  1. In any course, turn Edit Mode on and click 'Add an activity or resource'
  2. Select External tool
  3. Activity Name: (as appropriate, e.g. Class Poll - War of 1812)
  4. Preconfigured Tool: select ResponDog
  5. Click 'Select Content' and then click 'Continue' from the ResponDog popup window
  6. Configure Grade and Privacy settings as desired
  7. When you launch the assignment, you may use the links on the Instructor page to customize it for your class.

Installation instructions for PowerSchool Schoology

  1. Click the App Center icon, then select App Center.
  2. Locate ResponDog and click the title
  3. Click Install LTI 1.3 App > I Agree > Continue > Add to Organization
  4. You will be directed to Organization Apps to continue the installation.
  5. Click the Install/Remove button for ResponDog
  6. Select All Users, All Courses, or areas where you want to install ResponDog. Then Submit.
  7. Click Configure and copy the Deployment ID for ResponDog. It should be two 10-digit numbers separated by a hyphen. You will need it to complete the registration.
  8. Browse to https://respondog.com/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  9. Click the tokenized link in the email confirmation to register the Deployment ID and other configuration details with ResponDog. If the link expires, you can simply repeat the previous step to get another one.
The course instructor can then create any ResponDog assignment as follows:
  1. In any course, click 'Add Materials' and select ResponDog
  2. Click 'Continue' in the ResponDog popup window
  3. Use the Schoology assignment settings to edit the Grading Configuration and points
  4. When you launch the assignment, you may use the links on the ResponDog Instructor page to customize it for your class.
